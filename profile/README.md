# [Smitnner Studio](https://smitner.studio)


Indie game studio. Built on [Godot Engine](https://godotengine.org).

Currently shipping **[Hazard Pay](https://smitner.studio/hazardpay)** - a block-pushing puzzler set deep inside a secret, hazardous laboratory complex. Navigate the complex. Destroy the evidence. Cover your tracks.

---

## About

Small studio, strong opinions. Building games and the tooling around them - Godot addons, GDScript dev tools, and the odd experiment.
